Detailed Description

Tuples are core data structs used by NemaGUI API.

A major use of these data structs is in the event handling mechanism. Callback functions accept a void pointer in their arguments (signature contraint). Depending on the specific action that should take place during an event this void pointer is casted inside the implementation of each callback to specific data structs, most of which are defined here.
