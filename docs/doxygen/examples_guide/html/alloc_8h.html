Function: Test Harness free() 

     This works just like free() from the standard library.

Parameters: ptr - points to the block pointer returned by th_malloc(). file - the FILE macro from where the call was made line - the LINE macro from where the call was made

Note: This function is usually invoked by using th th_free() macro.

Note: Initial version of MITH relies on libc/OS to provide thread safe implementation for this function.

Note: It is valid to pass the null pointer to this function.
