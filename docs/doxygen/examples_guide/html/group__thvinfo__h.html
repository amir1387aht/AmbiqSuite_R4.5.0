#define EEMBC_TH_MAJOR   5

File Sentinel Version Text displayed to the user.

This is a human readable form of the version number. It is manually kept in sync with major, minor, step, and revision numbers. The revision minor, step, and revision are related to CVS tags. Major Release of Test Harness.
