◆ th_harness_poll()

int th_harness_poll ( void )

function: th_harness_poll

Description: Stub routine. Can be used for debugging.

This function can be called during the execution of a test or benchmark to give the test harness some execution time so it can respond to commands from a host.

Note: Only debug or test versions of true bench marks should call this function as it uses CPU time.

Returns: TRUE if the test should keep running. FALSE if the stop message has been received form the host and the benchmark or test should stop.
