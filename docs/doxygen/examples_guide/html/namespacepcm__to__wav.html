Detailed Description

Scripts to transform PCM data to WAV file.

Usage:
    pcm_to_wav.py [--input=<pcm_bin_file_name>]

Options:
    -h --help                                   Show this screen.
    --input=<pcm_bin_file_name>                 Input path of pcm data file.[default: .\record_data.bin]
