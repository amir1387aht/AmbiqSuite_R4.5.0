Detailed Description

Purpose: This example is the client (master) for the BLE Ambiq Micro Data Transfer Protocol. This example is meant to run on an Apollo4 EVB along with another Apollo4 EVB running ble_freertos_amdtps as the server. This example provides a UART command line interface with a simple menu that allows the user to scan, connect and initiate data transfers from either M->S or S->M direction.

Printing takes place over the ITM at 1M Baud.
